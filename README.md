# Infosec Rewind

Personal portfolio and cybersecurity blog by [Nikhil Kanade](https://github.com/M1cr0wave).

Built with React, styled with the Newsprint design system, and powered by Notion as a CMS.

- **React 19** + React Router
- **Framer Motion** for the dock and animations
- **Notion API** (`@notionhq/client`) — blog content is fetched at build time and baked into a static JSON file
- Deployed on **Vercel**
